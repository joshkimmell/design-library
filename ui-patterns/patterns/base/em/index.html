Emphasized Text

Emphasized text, <em>, is the emphasized phrase tag

Available Classes

Styling can be applied in the following ways:

  • .base--em - class
  • .base--STYLED em - <em> tag inside a class of .base--STYLED
  • %base--em - extended through Sass
