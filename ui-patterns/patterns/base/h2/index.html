Heading Level 2, <h2>, is a second level heading within an associated section

Available Classes

Styling can be applied in the following ways:

  • .base--h2 - class
  • .base--STYLED h2 - <h2> tag inside a class of .base--STYLED
  • %base--h2 - extended through Sass
