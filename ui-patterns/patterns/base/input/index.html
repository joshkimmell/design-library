Pattern Code

Basic Input

Basic input for a form.

Available Classes

Styling can be applied in the following ways:

  • .base--input - class
  • .base--STYLED input:not([type]) - <input> tag inside a class of .base--STYLED without any associated type
  • %base--input - extended through Sass
