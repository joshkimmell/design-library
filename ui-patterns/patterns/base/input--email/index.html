Email Input

Email input field for a form

Rendered Pattern

Pattern Code


<!-- {% pattern input from base %} -->
<label class="base--block-label" for="input1">Basic Email Input</label>
<input type="email" id="input1"   class="base--email-input" ></input>

<label class="base--block-label" for="input2">Placeholder Email</label>
<input type="email" id="input2" placeholder="email@example.com"   class="base--email-input" ></input>

<label class="base--block-label" for="input3">Disabled Email Input</label>
<input type="email" id="input3" placeholder="disabled-email@example.com"   class="base--email-input" disabled></input>


<!-- {% end pattern input from base %} -->

Email Input

Basic email input for a form.

Available Classes

Styling can be applied in the following ways:

  • .base--email-input - class
  • .base--STYLED input[type="email"] - <input type="email"> tag inside a class of .base--STYLED
  • %base--email-input - extended through Sass
